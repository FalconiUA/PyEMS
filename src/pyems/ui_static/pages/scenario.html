Scenario

Choose one active-power mode. PyEMS derives controller channels, safety claim, and allocation from the selected devices.

Unit envelope

The operating range and ramp limits of the controlled unit. These become the single allocation channel for the selected scenario — every setpoint is clamped into this envelope.

Export-spike protection

Caps how far the applied setpoint may sit above actual production, so power does not spike when the resource (sun, wind) suddenly returns.

Allocation priorities

When several controllers request the same setpoint, the allocator keeps the lower priority number. 0 is reserved for safety.

PID regulator

Tuning of the connection-point active-power regulator.

Generated bindings

These channel bindings are generated from the selected devices and written to site.yaml on save.

Required device registers

Each selected device profile must expose these fields. Open Device profiles to edit addresses, scales, access, and units.
