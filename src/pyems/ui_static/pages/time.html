Controller clock

This is the Raspberry Pi operating-system clock. It keeps running even when the EMS control loop is stopped; it is not taken from telemetry.

Clock policy

Define the local time basis independently from its source. This is the same separation used by industrial controllers: an IANA time zone carries legal seasonal rules; a fixed UTC offset never changes automatically.

Time source

Choose manual time or a daily synchronization with one NTP server.

Changing time does not require restarting the Raspberry Pi, the EMS runtime, or this web interface.

Synchronization diagnostics

The latest scheduled or manual NTP attempt is retained here with its exact outcome.
