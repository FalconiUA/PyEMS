Pi network address

This is the Pi's own address on the LAN — where you reach this console, and the source address it uses to talk to devices. It is not a Modbus device address (those live under Device profiles / Site YAML). For the EMS to reach a device directly, the Pi and the device must share one subnet.

Changing the address drops the connection you are using right now. After Apply, reopen this console at the new address — the page below will show the link.
